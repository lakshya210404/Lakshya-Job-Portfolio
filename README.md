# Lakshya Prasad — Developer Portfolio

Personal portfolio website showcasing my projects, technical skills, and experience as a software engineering student.

🔗 Live site: https://job-portfolio-murex.vercel.app

## Overview
This portfolio serves as a central place to highlight:
- Selected software engineering projects
- Technical skill set and tools
- Professional background and interests
- Contact and social links

The site is designed to be fast, clean, and responsive, with a focus on clarity and real-world engineering work.

## Features
- Modern responsive UI
- Project showcase with descriptions and links
- Deployed production build on Vercel
- Backend integration using Supabase
- SEO-friendly metadata and social sharing (OG images)

## Tech Stack
**Frontend**
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

**Backend / Services**
- Supabase

**Tooling & Deployment**
- Vercel
- ESLint
- PostCSS

## Project Structure
src/ → application source code
public/ → static assets
